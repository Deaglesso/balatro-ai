
# balatro-ai
AI project for Balatro

Installed mods: 
- Lovely Injector v0.9.0 (Latest Version) Mac ARM Version: https://github.com/ethangreen-dev/lovely-injector/releases/download/v0.9.0/lovely-aarch64-apple-darwin.tar.gz
- Steammods 1.0.0-beta-1224a (Balatrobot doesn't load in higher versions) : https://github.com/Steamodded/smods/archive/refs/tags/1.0.0-beta-1224a.tar.gz
- UV v0.11.8 (Latest Version): https://docs.astral.sh/uv
- Balatrobot v1.4.1 (Latest Version): https://github.com/coder/balatrobot/archive/refs/tags/v1.4.1.tar.gz


Use „uvx balatrobot serve” to start the game, and then run main.py via VS Code Debugger or with this command „python main.py”
